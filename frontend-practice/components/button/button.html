Semantic Icon Color Tokens

All button variants now use semantic icon color tokens. For proper icon color control, use inline SVG with fill="currentColor":

Uses --primary-icon-*
Uses --brand-icon-*
Uses --secondary-icon-*
Uses --naked-icon-*
Uses --outline-icon-*

Available Semantic Icon Tokens:

  • Primary: --primary-icon-default, --primary-icon-hover, --primary-icon-pressed, --primary-icon-disabled
  • Brand: --brand-icon-default, --brand-icon-hover, --brand-icon-pressed, --brand-icon-disabled
  • Secondary: --secondary-icon-default, --secondary-icon-hover, --secondary-icon-pressed, --secondary-icon-disabled
  • Naked: --naked-icon-default, --naked-icon-hover, --naked-icon-pressed, --naked-icon-disabled
  • Outline: --outline-icon-default, --outline-icon-hover, --outline-icon-pressed, --outline-icon-disabled
